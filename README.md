# Biblioteca UniFECAF

Este projeto é um site desenvolvido em HTML e CSS para representar a Biblioteca da UniFECAF.  
O objetivo é criar uma página institucional simples, organizada e responsiva, com informações sobre a biblioteca, livros disponíveis, unidades e contatos.

------------------------------------------------------------
 Acesso ao site
------------------------------------------------------------
O projeto está publicado com GitHub Pages e pode ser acessado pelo link:
https://marcosamambaia.github.io/bibilhotecaHTML/

------------------------------------------------------------
🗂 Estrutura do projeto
------------------------------------------------------------
bibilhotecaHTML/
├── index.html        # Página principal
├── css/
│   └── style.css     # Estilos da página
└── img/              # Imagens utilizadas no site

------------------------------------------------------------
📖 Seções do site
------------------------------------------------------------
- Banner principal: imagem de destaque da universidade
- Biblioteca: descrição da biblioteca e galeria de imagens
- Livros: cards com capas e descrições de livros
- Unidades: informações sobre diferentes unidades de ensino
- Contato: email, telefone e endereço para comunicação

------------------------------------------------------------
🛠 Tecnologias utilizadas
------------------------------------------------------------
- HTML5: estruturação do conteúdo
- CSS3: estilização e layout responsivo

------------------------------------------------------------
 Como clonar e rodar localmente
------------------------------------------------------------
# Clone o repositório
git clone https://github.com/marcosamambaia/bibilhotecaHTML.git

# Entre na pasta
cd bibilhotecaHTML

# Abra o index.html no navegador
